# Project Docs

This folder contains documentation for your project.

## Start Here

Copy the content from `AGENTS.md` in the root — it guides AI agents working on this codebase.

## Customizing This Template

- **docs/BUSINESS_IDEA.md** — Replace with your project's mission/mVP scope
- **flutter/README_*.md** — These are example docs showing how the Flutter app is structured
- **hermes-agent/skills/** — Pre-installed skills for the agent; browse and customize

## Typical Customization Steps

1. Update `AGENTS.md` with your project's mission and rules
2. Update `hermes-agent/config.yaml` with your API keys and settings
3. Update `hermes-agent/SOUL.md` with your agent's personality
4. Update `supabase/db/schema.ts` with your data model
5. Update `flutter/.env` and `supabase/db/.env` with your credentials
6. Update `firebase.json` with your Firebase project
